# Mouse-Dietary-Restriction-MRI

# Project Overview
The purpose of this project is to examine the long-term longitudinal effects that dietary restriction (defined as a reduction of 20-40% daily calories without malnutrition) has on brain structure and functional connectivity. Mice will be followed until 18-months old, and separated into three groups: ad libitum, dietary restriction, and high-fat diet. 

# MRI Acquisition
Mice will be scanned with the 7T BioSpec 70/20 small animal scanner at Academia Sinica, Taipei, Taiwan. Mice will be anesthetized with 0.025mg/kg dexedetomidine followed by 0.05mg/kg/hr and subjected to continuous inhalation of 4% isoflurane. Mice will undergo T2w, resting-state fMRI, and diffusion scans.

# Image Analysis
fMRI preprocessing will be conducted with the RABIES software by CoBrALab. Preprocessing of DTI images will be conducted with the Hou version of DSI Studio (Yeh, 2025)
